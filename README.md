# silentnomorechina.com

An informational site focused on ending domestic violence in China. It aggregates resources, global best practices, guidance on supporting survivors, personal stories, and a news log.

## Sections

- Resources (Help in China)
- Success Stories (Abroad)
- Support (How to Support a Survivor)
- Global Protective Measures
- Personal Stories
- News & Case Log

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS
- GSAP animations

## Local Development

```bash
npm install
npm run dev
# build
npm run build
```

## Deploy (Vercel)

- vercel.json is configured: `buildCommand: npm run build`, `outputDirectory: dist`, `framework: vite`.
- Connect the GitHub repo in Vercel, import, and deploy.
